# BankUP official repository

## How to run this project?
### 1. Clone this repository
Using this command: 
```bash
 git clone git@github.com:Kepalez/BankUP.git
```
On a safe folder
### 2. Install all dependencies
Using this command: 
```bash
 npm install
```
### 3. Run the server
After installing all dependencies now you can run:
```bash
 npm run start
```
or 
```bash
 npm run web
```
